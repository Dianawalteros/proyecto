#  Sistema de Gestión para Empresa de Dulces

##  Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación web para la gestión de una empresa de dulces.
El sistema permite administrar productos, registrar usuarios, gestionar ventas y visualizar la información de la tienda de manera organizada.

El objetivo principal del sistema es facilitar el control y administración de los productos y ventas de la empresa, permitiendo una mejor gestión del negocio.

---

## ?? Objetivos

### Objetivo General

Desarrollar un sistema web que permita gestionar los productos, usuarios y ventas de una empresa de dulces.

### Objetivos Específicos

* Registrar y administrar productos.
* Gestionar usuarios del sistema.
* Permitir la compra de productos.
* Visualizar el carrito de compras.
* Controlar el registro de ventas.

## ?? Instalación del Proyecto

1. Clonar el repositorio

```
git clone https://github.com/Dianawaleteros/proyecto.git
```

2. Entrar a la carpeta del proyecto

```
cd proyecto
```

3. Instalar dependencias

```
npm install
```

4. Ejecutar el servidor

```
npm start
```
